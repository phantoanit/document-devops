Distributed mode / phan tan luu tru / 4 node (tối thiểu production)

Erasure Coding (EC)
Ví dụ cluster bạn: 4 node × 4 disk = 16 disk
MinIO sẽ: Chia object thành:
8 data blocks
8 parity blocks

Ghi lên 16 disk
👉 Bạn có thể mất:
1 node
hoặc nhiều disk
mà dữ liệu vẫn đọc được

Thành phần	Khuyến nghị
CPU	≥ 8 core (physical)
RAM	≥ 32 GB
Disk	4–8 SSD/NVMe giống nhau
Network	Dual NIC nếu có
OS Ubuntu 22.04 / RHEL 8+

Port	Mục đích
9000	S3 API
9001	Console

#IAM & phân quyền
#MinIO có sẵn các nhóm quyền cơ bản mà bạn có thể gán ngay lập tức cho User:
readonly: Chỉ được xem và tải file.
readwrite: Được phép đọc, ghi và xóa file trên tất cả các bucket.
diagnostics: Quyền xem thông tin hệ thống.
writeonly: Chỉ được phép upload.

mc admin policy attach onprem readwrite --user appuser

# Gan quyen cu the cho bucket, bat buoc tao policy
Chỉ cho phép Download (Read-Only): Thay s3:* thành ["s3:GetObject", "s3:GetBucketLocation"].
Chỉ cho phép Upload (Write-Only): Thay s3:* thành ["s3:PutObject"].
Toàn quyền (Admin của Bucket đó): Dùng s3:* như ví dụ trên.
vi my-policy.json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": ["s3:GetBucketLocation", "s3:ListBucket"],
      "Resource": ["arn:aws:s3:::app-data"]
    },
    {
      "Effect": "Allow",
      "Action": ["s3:*"],
      "Resource": ["arn:aws:s3:::app-data/*"]
    }
  ]
}
# Thêm policy vào hệ thống
mc admin policy create onprem my-bucket-full-access my-policy.json

upstream minio_api {
    least_conn;
    server 10.60.10.30:9000 max_fails=3 fail_timeout=10s;
    server 10.60.10.31:9000 max_fails=3 fail_timeout=10s;
    server 10.60.10.32:9000  max_fails=3 fail_timeout=10s;
    server 10.60.10.33:9000 max_fails=3 fail_timeout=10s;
}

server {

    listen 443 ssl;
    server_name minio.abc.vn; 
    ssl_certificate /etc/letsencrypt/live/wildcard.abc.vn/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/wildcard.abc.vn/privkey.pem;
    # --- Cấu hình tối ưu cho Upload/Download ---
    ignore_invalid_headers off;
    client_max_body_size 0;
    proxy_buffering off;
    proxy_request_buffering off;

    ssl_protocols TLSv1.2 TLSv1.3;

    location / {
        proxy_set_header Host $http_host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_http_version 1.1;
        proxy_set_header Connection "";
        chunked_transfer_encoding on;
        proxy_connect_timeout 300;
        proxy_send_timeout 300;
        proxy_read_timeout 300;
        proxy_pass http://minio_api;
    }
}
